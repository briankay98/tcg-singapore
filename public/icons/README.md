# PWA Icons

This directory contains Progressive Web App icons for TCG Singapore.

## Required Sizes:
- 72x72px - Small icon
- 96x96px - Medium icon  
- 128x128px - Large icon
- 144x144px - Extra large icon
- 152x152px - iPad icon
- 192x192px - Android icon
- 384x384px - Large Android icon  
- 512x512px - Splash screen icon

## Design Guidelines:
- Use the TCG Singapore red color (#dc2626)
- Include Singapore flag emoji or elements
- Ensure icons work on both light and dark backgrounds
- Follow maskable icon guidelines for Android

For now, placeholder icons are used. Replace with actual designed icons for production.
